# SimpleUI for KOReader

A simple and clean UI plugin for KOReader. Get quick access to Home, History, Collections, and your custom folders right from the bottom of your screen.

## Features

- Bottom navigation bar with customisable tabs
- Top status bar (clock, battery, frontlight)
- Desktop widget with Quote of the Day
- Reading goals tracker
- Collections widget

## Installation

1. Download this repository as a ZIP (click **Code** → **Download ZIP**)
2. Extract the folder and rename it to `simpleui.koplugin` if needed
3. Copy the folder to the `plugins/` directory on your KOReader device
4. Restart KOReader
5. Go to **Menu → Tools → SimpleUI** to configure
